## Описание программы

Это простое приложение для учета личных расходов с графическим интерфейсом. Данные сохраняются в файл `data.json`, что позволяет не терять историю между запусками программы.

## Как использовать

1. Запустите файл `main.py`.
2. В полях ввода укажите:
   - **Описание** — что купили.
   - **Категория** — к какой группе расходов относится (Еда, Транспорт и т.д.).
   - **Сумма** — стоимость в рублях (только числа).
3. Нажмите кнопку **«Добавить расход»**.
4. Все записи отобразятся в таблице ниже.

## Примеры использования

- **Покупка:** Кофе в кафе.
- **Категория:** Еда.
- **Сумма:** 350.
- **Результат:** Запись появится в таблице с текущей датой.

## Требования

- Python 3.x
- Библиотеки: `tkinter`, `json` (входят в стандартную библиотеку Python).
